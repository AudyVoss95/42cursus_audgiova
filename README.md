# 42 Cursus — Software Engineering Projects 🚀

Repository containing the core projects developed during the Software Engineering curriculum at **42 School**. Focus on low-level C programming, algorithm optimization, Unix environment, and strict code compliance (Norminette).

---

## 📌 Projects Index

| Project | Language | Description | Key Concepts |
| :--- | :---: | :--- | :--- |
| **libft** | `C` | Custom C standard library re-implementation. | Memory allocation, pointers, string handling, data structures. |
| **ft_printf** | `C` | Custom implementation of the standard `printf` function. | Variadic functions (`va_list`), format specifiers, type conversion. |

---

## 🛠️ Tech Stack & Standards
- **Language:** C (C99 / C11)
- **Compiler:** `gcc` / `clang` with flags `-Wall -Wextra -Werror`
- **Build System:** `Makefile` (all, clean, fclean, re)
- **Coding Standard:** 42 Norminette compliance

---

## 💻 How to Build and Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AudyVoss95/42cursus_audgiova.git
   cd 42cursus_audgiova
   ```

2. **Navigate to a specific project (e.g., libft):**
   ```bash
   cd Libft
   make
   ```
